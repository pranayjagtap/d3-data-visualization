Homework #2 D3 Programming

Title: AUS Open Stats

Visualization of Australian open data using D3 to encode features of data in order to gain insights.

Designed by Pranay Jagtap

Intrduction

D3 library is used to encode data into below visualizations.

Features and interactions:

  • Filter Interaction 1: Data by year- Select a year to filter data by year
  • Interaction 2: See popup annotations on mouse hover over circle in scatter plot
  • Filters: Filter the overview with respect to year and/or round of play using the two drop down provided.
  • Interaction 3: Click on individual circles to generate second visualization which is drilled down to a detait view of players with whom that particular circle corresponds to.
